# BoardGameHub

BoardGameHub is a Django web application for browsing board games, managing profiles, writing reviews, and organizing events.

## Features
- User registration, login, and logout
- Extended user profile
- Board game catalog
- Reviews system
- Events app
- Django admin panel
- Groups and permissions

## Tech Stack
- Python 3.13
- Django 5.x
- SQLite for local development

## Setup
1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Apply migrations: py manage.py migrate
4. Create superuser: py manage.py createsuperuser
5. Run server: py manage.py runserver

## Environment Variables
No extra environment variables are currently required for local development.

## Deployment
The project is prepared for future cloud deployment.

## Project Idea
This project was created as part of the Django Advanced course.
The goal is to build a board game community platform where users can browse games, write reviews, and organize board game events.
